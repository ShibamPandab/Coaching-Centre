# Aurum Academy

## Overview

**Aurum Academy** is a premium, multi-page website concept for an elite coaching institute specializing in **IIT-JEE, NEET, and Olympiad** preparation. Built as a **frontend showcase**, it presents the academy's courses, faculty, results, and admissions journey through a luxury dark-and-gold interface with smooth, animated navigation across multiple pages.

---

## Features

- 🎓 **Multi-Page Architecture** — dedicated routes for Home, About, Courses, Faculty, Results, Gallery, and Contact using React Router
- 🏆 **Animated Statistics Counters** — highlights like top exam ranks, percentile scores, and alumni count
- 💎 **"Aurum Advantage" Pillars** — feature cards outlining the institute's pedagogy, mentorship, and mock-exam systems
- 🎯 **Scholarship Admission Test (ASAT) Banner** — call-to-action section promoting the academy's scholarship program
- 📚 **Courses Page** — detailed breakdown of programs with highlights for each course
- 👩‍🏫 **Faculty Page** — profile cards introducing the academy's educators
- 📈 **Results Page** — topper showcases with category tabs and student testimonials
- 🖼️ **Gallery Page** — filterable category-based image gallery
- 📞 **Contact Page** — institute contact details and enquiry information
- 💬 **Floating WhatsApp Widget** — persistent quick-contact button across all pages
- 🌀 **Smooth Page Transitions** — animated route changes with automatic scroll-to-top

## Installation

```bash
# Clone the repository
git clone https://github.com/ShibamPandab/Coaching-Centre.git

# Move into the project directory
cd Coaching-Centre

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173` by default.
